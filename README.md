# EscapeRoom‑Vizard7  
Vienkārša 3D “escape room” spēle, uzbūvēta ar Vizard 7

## Par projektu  
Šī ir eksperimentālā spēle — telpa, kurā spēlētājs var brīvi pārvietoties, apskatīt objektus un mijiedarboties ar tiem. Piemēram, nospiežot uz gleznas pie sienas, tā var pastumt malā un atklāt slepenu seifu vai telpu aiz sienas. Projekts ir veidots, lai iemācītos, kā importēt 3D vidi, padarīt objektus interaktīvus un realizēt vienkāršu “slēpņu un atklājumu” mehāniķi.

## Prasības  
- Instalēts Vizard 7  
- Python 3.x 

## Kā palaist  
1. Klonē repozitoriju:  
- ielādē .zip ar visiem failiem
- "extract" failus vienā mapē
- atver Vizard7 un atver .py failu
- piespied zaļo bultiņu.
